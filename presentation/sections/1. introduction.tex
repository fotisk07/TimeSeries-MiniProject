\begin{frame}{Motivation}

\textbf{Problem.}
Team formations in soccer are \emph{fluid but structured}: players temporarily swap roles (pressing, overlaps), while genuine tactical changes occur at specific moments.
\vspace{0.3cm}

\textbf{Solution.}
We study \textbf{SoccerCPD}~\cite{Kim2022SoccerCPD}, a change-point detection framework that segments a match into tactically consistent periods by:
\begin{enumerate}
    \item detecting formation changes from role-adjacency graphs (FormCPD)
    \item detecting role switches via permutation dynamics (RoleCPD).
\end{enumerate}

% \vspace{0.3cm}
\end{frame}

\textbf{Our Contributions (Beyond the Paper).}
\begin{itemize}
    \item \textbf{Context-aware formations:} separation of attacking vs.\ defensive shapes using possession.
    \item \textbf{Role stationarity metric:} quantifying player role stability (fixed vs.\ roaming).
    \item \textbf{Data generalization:} adaptation of the framework to sparse event-based data (StatsBomb).
\end{itemize}
